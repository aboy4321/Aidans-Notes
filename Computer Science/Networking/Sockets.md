### Definition
A *socket* is an endpoint of a two way communication link between two programs running on a network. 
### Uses
*Socket* provides a means of establishing communication by creating named contact points which the communication takes place. (Essential for a [[Web Server]]).
### Example
*A Stream Socket* is a type of network socket that provides a connection and unique flow of data without record boundaries with well defined mechanisms for creating and destroying connections and for detecting errors.
